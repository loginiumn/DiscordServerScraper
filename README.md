# **DiscordServerScraper**

>## DESCRIPTION  


 A browser-based tool designed to export Discord server member lists using Discord token authentication. It’s still a work in progress, so if you’re looking to scrape member lists, IT WILL NOT WORK. 



>## FEATURES 

 This browser-based tool is designed to scrape and export Discord server member data. It allows users to extract  information including usernames, user IDs (UIDs), dates of joining (DOJ), roles, and display names for members of any Discord server they have access to.



 The tool features a light and dark mode for users to use and supports exporting data in various file formats like ``JSON``, ``.HTML``, .``CSV,`` ``.TXT``, ``.I64`` format. It includes a profile display and a server listing feature. 


>## GUIDES
- Read the [Token.md](https://github.com/loginiumn/DiscordServerScraper/blob/main/Token.md)
- This will guide you on how to get your Discord token. If you have a Discord account, you can start by joining random servers to test if the tool works. Keep in mind, this tool is still a work in progress.
